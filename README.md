# 🐺 人狼ゲーム (小学生・タブレット向けWebアプリ)

1台のタブレットやスマートフォンをみんなで回して遊べる、小学生向けに最適化された人狼ゲームのWebアプリケーションです。 ブラウザ（フロントエンド）のみで動作するため、サーバー環境は不要。GitHub Pagesなどでホスティングするだけですぐに遊べます。

## ✨ 特徴・機能

*   🏫 **小学生向けの究極ストレスフリー設計**
    
    *   表示されるすべての漢字にふりがな（ルビ）を完備。
        
    *   名前入力を省略しても自動で「プレイヤー1」などが割り当てられ、ワンタップですぐにゲームを開始できます。
        
*   📱 **1台の端末でオフラインプレイ**
    
    *   複数人で1台のタブレットを回して遊ぶプレイスタイルに完全特化。
        
*   🔊 **AIゲームマスター（自動音声・SE機能）**
    
    *   Web Speech APIを活用し、夜の進行や追放時のアナウンスを自動で読み上げます。
        
    *   操作音や昼夜の切り替えジングルなど、臨場感を高める効果音（SE）を内蔵しています。
        
*   🎨 **リッチなUI/UXとアニメーション**
    
    *   昼は青空と雲、夜は星空へとシームレスに変化するアニメーション背景。
        
    *   押し心地の良い3Dボタンやグラスモーフィズム（すりガラス）パネルを採用し、商用アプリのようなリッチな体験を提供します。
        
*   ⚙️ **役職の自由カスタマイズ**
    
    *   プレイ人数（4〜12人）に合わせて、役職（村人、人狼、占い師、霊能者、狩人、狂人）の内訳を自由に調整可能です。
        

## 🎮 遊び方

1.  **ゲームの設定**: 遊ぶ人数を選び、役職の内訳をカスタマイズして「ゲーム開始」を押します。
    
2.  **役職の確認**: 画面の指示に従ってタブレットを回し、自分の役職をこっそり確認します。
    
3.  **昼の話し合い**: タイマー（1分/3分/5分）を使って、誰が人狼か話し合います。
    
4.  **投票**: 追放したい人を順番に選びます（最多票の人が追放されます。同数の場合は決選投票またはランダム）。
    
5.  **夜の行動**: 人狼は襲撃し、占い師や狩人などは特別な能力を使います。
    
6.  **勝敗**: 人狼を全滅させるか、人狼の数が村人陣営と同数以上になるまでゲームを繰り返します。
    

## 🛠️ 技術スタック

*   **フレームワーク**: React (Hooksを活用した完全な状態管理)
    
*   **スタイリング**: Tailwind CSS (HTML内にスタイルを統合)
    
*   **アイコン**: Lucide React
    
*   **音声・音響**: Web Speech API (読み上げ), Web Audio API (SE生成)
    

## 🚀 公開・セットアップ方法

このアプリは静的なWebページとして動作します。

1.  このリポジトリのファイル一式をダウンロードまたはクローンします。
    
2.  GitHub Pages など、静的ファイルをホスティングできるサービスにアップロードします。
    
3.  （オプション）リポジトリのルートディレクトリに `favicon.png` を配置すると、ブラウザのタブにアイコンが表示されます。
